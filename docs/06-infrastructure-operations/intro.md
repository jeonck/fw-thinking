# 인프라, 운영 및 신기술 대응

현대 IT 인프라는 클라우드 네이티브 전환과 함께 **탄력성(Resilience)**과 **자동화(Automation)**가 핵심 가치가 되었습니다.

## 핵심 프레임워크

| 프레임워크 | 설명 |
|---|---|
| **ITIL 4** | IT 서비스 운영 관리 |
| **SRE** | 사이트 신뢰성 엔지니어링 |
| **Cloud Adoption Framework** | 클라우드 전환 방법론 |
| **IaC** | 인프라 as Code |

## 학습 포인트

- 하이브리드·멀티 클라우드 전략
- 컨테이너/쿠버네티스 운영
- 관찰 가능성(Observability) 체계
- 재해복구(DR) 및 BCP 수립
