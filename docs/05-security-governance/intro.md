# 정보보호 및 보안 거버넌스

정보보호는 기밀성(Confidentiality), 무결성(Integrity), 가용성(Availability)의 **CIA 트리아드**를 중심으로 체계적으로 관리됩니다.

## 핵심 프레임워크

| 프레임워크 | 설명 |
|---|---|
| **ISO/IEC 27001** | 정보보호 관리체계(ISMS) |
| **NIST CSF** | 사이버보안 프레임워크 |
| **Zero Trust** | 제로 트러스트 보안 모델 |
| **ISMS-P** | 개인정보보호 관리체계 |

## 학습 포인트

- 위험 관리(Risk Management) 절차
- 침해사고 대응(IR) 프로세스
- 클라우드 보안 아키텍처
- DevSecOps 보안 내재화
