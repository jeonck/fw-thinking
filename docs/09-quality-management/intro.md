# 품질 관리 및 신뢰성

소프트웨어 품질은 **기능적 적합성, 성능 효율성, 보안성** 등 다차원적 속성을 통해 측정됩니다.

## 핵심 프레임워크

| 프레임워크 | 설명 |
|---|---|
| **ISO/IEC 25010** | 소프트웨어 품질 모델 |
| **Six Sigma** | 데이터 기반 품질 개선 방법론 |
| **TQM** | 전사적 품질 관리 |
| **SLA/SLO/SLI** | 서비스 수준 관리 체계 |

## 학습 포인트

- 소프트웨어 테스트 전략(V-모델)
- 기술적 부채와 리팩토링
- 성능 테스트 및 부하 테스트
- 신뢰성 공학(Reliability Engineering)
