# 데이터 관리 및 AI 거버넌스

데이터는 현대 디지털 전환의 **핵심 자산**이며, AI 시대에는 데이터 품질과 거버넌스가 경쟁력의 원천입니다.

## 핵심 프레임워크

| 프레임워크 | 설명 |
|---|---|
| **DAMA-DMBOK** | 데이터 관리 지식 체계 |
| **DCAM** | 데이터 관리 성숙도 모델 |
| **MLOps** | AI/ML 운영 방법론 |
| **EU AI Act** | AI 규제 및 거버넌스 체계 |

## 학습 포인트

- 데이터 레이크 vs 데이터 웨어하우스
- 마스터 데이터 관리(MDM)
- AI 윤리와 설명 가능한 AI(XAI)
- 데이터 메시(Data Mesh) 아키텍처
