# 엔터프라이즈 아키텍처 (EA) 및 설계

EA는 조직의 비즈니스 목표와 IT 시스템 간의 **정렬(Alignment)**을 체계적으로 관리하는 설계 방법론입니다.

## 핵심 프레임워크

| 프레임워크 | 설명 |
|---|---|
| **TOGAF** | The Open Group Architecture Framework |
| **Zachman** | 엔터프라이즈 아키텍처 분류 체계 |
| **DoDAF** | 미국 국방부 아키텍처 프레임워크 |
| **FEA** | 연방 엔터프라이즈 아키텍처 |

## 아키텍처 도메인

- **비즈니스 아키텍처**: 전략, 프로세스, 조직
- **데이터 아키텍처**: 데이터 구조 및 관리
- **애플리케이션 아키텍처**: 시스템 및 통합
- **기술 아키텍처**: 인프라 및 플랫폼
