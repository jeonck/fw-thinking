# IT 거버넌스 및 전략 경영

IT 거버넌스는 조직이 IT를 통해 전략적 목표를 달성하고 리스크를 관리하기 위한 **의사결정 체계와 책임 구조**입니다.

## 핵심 프레임워크

| 프레임워크 | 설명 |
|---|---|
| **COBIT** | IT 거버넌스 및 관리를 위한 종합 프레임워크 |
| **ITIL** | IT 서비스 관리 모범 사례 체계 |
| **ISO/IEC 38500** | IT 거버넌스 국제 표준 |
| **BSC** | 균형 성과표(Balanced Scorecard) |

## 학습 포인트

- IT 거버넌스와 IT 관리의 차이
- 이해관계자 중심의 가치 창출
- 리스크 관리와 자원 최적화
- 전략-실행 정렬(Strategic Alignment)
