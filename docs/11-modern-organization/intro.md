# 현대적 조직 및 일하는 방식

디지털 시대의 조직은 **민첩성(Agility)**과 **협업(Collaboration)**을 중심으로 운영 방식을 혁신하고 있습니다.

## 핵심 프레임워크

| 프레임워크 | 설명 |
|---|---|
| **SAFe** | 기업 규모의 애자일 프레임워크 |
| **OKR** | 목표와 핵심 결과 지표 |
| **Spotify Model** | 부족-분대 조직 모델 |
| **Digital Transformation** | 디지털 전환 성숙도 모델 |

## 학습 포인트

- 애자일 조직 설계 원칙
- 원격·하이브리드 근무 모델
- 지식 관리와 학습 조직
- 변화 관리(Change Management)
